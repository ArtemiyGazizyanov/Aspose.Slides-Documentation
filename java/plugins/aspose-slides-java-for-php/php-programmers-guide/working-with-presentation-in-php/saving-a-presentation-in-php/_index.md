## **Aspose.Slides - Saving a Presentation**
In order to save presentation using *Aspose.Slides Java for PHP{*}, you can use following code.

**PHPCode**

```



$save_format = new SaveFormat();

pres->save($dataDir . "output.pptx", $save_format->Pptx)

```
